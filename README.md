# Personal Website

A personal website for Timileyin Agbaje, a Computer Science undergraduate interested in web technologies and their integration with AI systems.

Built with HTML and CSS, the site serves as a simple landing page with a short bio, social links, and contact information. Deployed via GitHub Pages.

## Responsive Notes

- Updated layout to remove fixed top spacing and use `min-height` with safe viewport units for better behavior on small-screen smartphones.
- Added padding and link wrapping so content no longer overflows on narrow devices (320px and up).
