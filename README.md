## Getting Started

No build tools or package managers are required. The project is plain HTML/CSS/JS.

### View Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/Timileyin-Agbaje/Personal-Website.git
   cd Personal-Website
   ```

2. **Open in a browser**

   Simply open `index.html` directly in your browser:

   ```bash
   # macOS
   open index.html

   # Linux
   xdg-open index.html

   # Windows
   start index.html
   ```

   Or use a local development server for a more realistic environment:

   ```bash
   # Python 3
   python -m http.server 8080
   # then visit http://localhost:8080
   ```

## Deployment

The site is deployed via **GitHub Pages**. Any push to the default branch will automatically update the live site. The `CNAME` file configures the custom domain.
